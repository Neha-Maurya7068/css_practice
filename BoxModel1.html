✅ What is the Box Model?

The CSS Box Model is a layout mechanism that treats every HTML element as a rectangular box.
This box contains content, padding, border, and margin.


Parts of the Box Model

1️⃣ Content

  • The actual text, image, or element inside the box.
  • Size is controlled by:
  • width , height

2️⃣ Padding

  • Space between content and border.
  • Provides inner spacing.
  • Background color applies to padding also.
  • Example:-
  • padding: 20px;

3️⃣ Border

  • A line around the padding and content.
  • Example:
  • border: 2px solid black;

4️⃣ Margin

  • Space outside the border.
  • Creates distance between elements.
  • Transparent (no color).
  • Example:
  • margin: 10px;
