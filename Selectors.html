CSS Selector

Selector:-

A CSS Selector is a pattern used to select (target) the HTML elements on which you want to apply CSS styles.

Types of CSS Selector:--

1. Universal Selector (*):-

Selects all elements.

Example:-
*{
margin: 0;
padding: 0;
}

2. Element Selector (Tag Selector):-

Selects HTML tags.

Example:-
h1 {
color: red;
}

3. Class Selector (.):-

Selects elements with a class attribute.

Example:-
.title {
font-size: 30px;
}

4. ID Selector (#):-

Selects elements with a unique id.

Example:-
#header {
background: yellow;
}

5. Attribute Selector:-

Selects elements based on attributes.

Example:-
input[type="text"] {
border: 2px solid blue;
}

6. Group Selector (, ,):-

Apply same style to multiple selectors.

Example:-
h1, h2, p {
color: green;
}

7. Descendant Selector (Space ):-

Selects elements inside another element.

Example:-
div p {
color: purple;
}

8. Child Selector (>):-

Selects direct child only.

Example:-
ul > li {
color: brown;
}

9. Adjacent Sibling Selector (+):-

Selects the next sibling.

Example:-
h1 + p {
color: orange;
}

10. General Sibling Selector (~):-

Selects all siblings after an element.

Example:-
h1 ~ p {
color: pink;
}

11. Pseudo-class Selector (:):-

Selects elements in a special state.

Example:-
p::first-letter {
font-size: 40px;
}

12. Pseudo-element Selector (::):-

Selects part of an element.

Example:-
p::first-letter {
font-size: 40px;
}
